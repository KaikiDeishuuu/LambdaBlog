## 嵌入式 React 组件测试

接下来是 MDX 的核心功能：在内容中无缝使用 React 组件。

### 无状态组件：`<Highlight />`

我们可以用刚刚定义的组件来 <Highlight>高亮重要的文本</Highlight>。由于它是一个 React 组件，我们可以轻松地给它传递 props，比如改变颜色：<Highlight color="#bae6fd">像这样变成蓝色</Highlight>。

### 有状态组件：`<Counter />`

下面是一个更复杂的例子，它拥有自己的内部状态。你可以点击按钮来与它互动，这个状态完全由组件自己管理，独立于页面的其他部分。
