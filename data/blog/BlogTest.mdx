## 什么是 MDX？

MDX 是一种 Markdown 的超集，它让你能够在 Markdown 内容中直接使用 JSX 语法。这意味着你可以像编写标准 Markdown 一样编写内容，同时也能随时引入交互式的 React 组件。

这是一个普通段落。你可以使用*斜体*、**粗体**，或者 `行内代码`。

### 主要特性

- **Markdown 语法**: 支持所有你熟悉的 Markdown 功能。
- **JSX 支持**: 可以直接在文件中编写和导入 React 组件。
- **组件导入/导出**: 可以像在 JavaScript 模块中一样导入和导出数据或组件。
- **强大的交互性**: 为你的静态内容增添动态功能。
